# Dataset - Statistiques des Joueurs des 5 grands championnats de football européens

## Contexte

Ce dataset rassemble les statistiques de performance des Joueurs des 5 grands championnats de football européens. Il s'agit de données détaillées permettant d'analyser les contributions individuelles des joueurs à travers différents aspects du jeu.

## Contenu

Le dataset comprend des métriques variées qui couvrent l'ensemble du spectre de performance d'un joueur de football :

Les **statistiques offensives** incluent les buts marqués, les expected goals (xG), les tirs, ainsi que les passes décisives et leur valeur attendue. Ces données permettent d'évaluer la contribution offensive directe de chaque joueur.

Les **statistiques de jeu et de création** regroupent les informations sur les passes, leur précision, ainsi que les actions progressives avec et sans ballon. On y trouve également les données sur les dribbles et les touches dans les zones dangereuses.

Les **statistiques défensives** couvrent les interventions défensives comme les tacles, les interceptions, les blocages de balles, les dégagements et les duels aériens remportés.

## Joueurs

Le dataset présente sept joueurs occupant différents postes sur le terrain : défenseurs centraux, latéraux, milieux défensifs et centraux, ailiers et avant-centre. Cela permet d'avoir un aperçu représentatif des différents profils et rôles au sein d'une équipe.

## Objectif

Ces données permettent d'analyser et de comparer les performances individuelles, d'identifier les points forts de chaque joueur selon son poste, et de comprendre comment chacun contribue au jeu collectif de l'équipe.
